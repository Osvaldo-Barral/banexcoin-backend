# 💰 BbanexCoin API

API REST desarrollada con **NestJS**, **TypeORM** y **PostgreSQL** para la gestión de cuentas, transacciones y comisiones dentro de un sistema de billetera digital.

---

## 📦 Requisitos

- Node.js >= 18.x
- NPM >= 9.x 
- PostgreSQL >= 16
- NestJS CLI (`npm install -g @nestjs/cli`)

---
